# Sistema de Gestión de Estudiantes (archivos binarios) – C

¡Bienvenido al proyecto aplicación de consola en C para registrar, buscar y listar estudiantes usando **archivos binarios**.

## Requisitos
- GCC / MinGW / WSL / Code::Blocks / Dev-C++ / Replit / JDoodle
- Git / GitHub para versionado

## Estructura
- `src/main.c` : código fuente
- `data/estudiantes.dat` : archivo binario (se genera al ejecutar)
- `docs/` : informe y capturas

## Uso
- Desarrollar una aplicación de consola en C que gestione estudiantes usando archivos binarios:
- Registro
- Búsqueda
- Visualización
- Y opcionalmente Modificación y Baja Lógica

##Ejecución
- Windows: gestion_estudiantes.exe
- Linux/Mac/WSL: ./gestion_estudiantes

## Compilación
```bash
gcc src/main.c -o gestion_estudiantes
